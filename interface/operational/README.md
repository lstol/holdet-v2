# interface/operational/

Phase 6 — Full operational dashboard.

## Views
- Pre-stage briefing (EV ranking, captain recommendation, transfer suggestions, odds divergence panel)
- Live diagnostic mode (informational re-executions from fresh Race-State snapshots — own execution boundary per run)
- Post-stage review (predicted vs actual, calibration log)
- Expert knowledge capture (team orders, form signals — feeds override layer, not Layer 0)

## Live diagnostic mode constraint
Each re-execution in live mode has its own execution boundary and GBC reference.
Outputs are informational only unless explicitly authorized by governance rules.
