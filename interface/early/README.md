# interface/early/

Phase 3 — Early inspection interface. Priority: visibility before correctness.

## Views
- Rider card view (Layer 0 attributes, provenance, data age)
- Stage profile viewer (elevation, classification, TTT flag, weather)
- Interaction output inspector (terrain fit diagnostics, fatigue indicators)
- Naive probability display (baseline model output, inspectable)
- EV table per rider per stage (line-item breakdown)
- Override panel (log probability adjustments with reason and source)

## Constraint
No computation in the interface layer. All data consumed from engine outputs.
